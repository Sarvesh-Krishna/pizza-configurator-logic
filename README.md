# 🍕 Pizza Configurator Logic

This is a foundational Python project designed to practice *Conditional Flow Control* and *User Input Handling*. It guides a user through a step-by-step process to build a custom pizza order.

## 🧠 Coding Concepts Used
- **User Inputs:** Gathering strings from the console to drive program flow.
- **Conditional Logic:** Using `if-else` statements to handle different selection paths.
- **String Formatting:** Displaying a personalized final summary of the user's choice.

## 🛠️ How it Works
The program follows a linear selection process:
1. **Portion Selection:** Choose between "Personal" or "Family" sizes.
2. **Sauce & Cheese:** Interactive prompts for sauce and cheese preferences.
3. **Toppings:** final step to add a custom topping.
4. **Summary:** Prints a final confirmation of exactly what was selected.
